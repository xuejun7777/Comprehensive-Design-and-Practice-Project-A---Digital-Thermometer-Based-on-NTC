# NTC数字温度计综合设计项目

## 项目简介

基于STM32G030微控制器和NTC热敏电阻的数字温度计系统，包含完整的硬件设计、软件开发和PCB制作。

## 系统架构

### 硬件组成
- **MCU**: STM32G030 (ARM Cortex-M0+)
- **传感器**: NTC热敏电阻 (10kΩ@25°C, B=3950K)
- **显示**: 4位7段数码管
- **接口**: USB Type-C + UART

### 软件功能
- 12位ADC温度采集
- Steinhart-Hart方程温度转换
- 动态数码管显示
- 串口数据输出
- 开机学号显示

## 核心特性

- **通信**: 115200bps UART

## 关键算法

### 温度转换公式
```c
// Steinhart-Hart方程
T = 1/((ln(Rt/R0)/B) + (1/T0)) - 273.15
```
